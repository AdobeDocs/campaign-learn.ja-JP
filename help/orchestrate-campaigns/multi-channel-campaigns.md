# マルチチャネルキャンペーンの作成

マルチチャネルキャンペーンでは、単一のマーケティングコミュニケーションが複数のチャネルを使用して、受信者の優先チャネルに基づいて、より広いターゲットオーディエンスに到達します。

マルチチャネルの配信は、単一のワークフローで設計および実行されます。

Eメール、SMS、ダイレクトメール配信を使用してマルチチャネルキャンペーンを作成する方法を説明します。
